
# AWS Cloud Engineering Labs

## Overview
This repository contains hands-on AWS cloud projects demonstrating practical experience with core cloud services including EC2, S3, IAM, and storage solutions. Each lab focuses on real-world deployment, configuration, and validation.

---


## Projects Included

### 1. EC2 Apache Web Server Deployment
- Launched and configured an EC2 instance
- Installed and configured Apache web server
- Enabled public access via security groups
- Verified deployment through browser access

### 2. S3 Static Website Hosting
- Created and configured S3 bucket for static hosting
- Enabled public access and bucket policies
- Deployed website files and tested accessibility

### 3. EBS Volume Management
- Attached and mounted EBS volumes to EC2
- Configured persistent storage in Linux
- Verified data persistence after reboot
