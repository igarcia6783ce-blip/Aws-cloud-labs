AWS S3 Static Website Lab

## Overview
This project demonstrates hosting a static website using Amazon S3.

## Live Website
http://ivan-s3-lab-static-site.s3-website-us-east-2.amazonaws.com/

## AWS Services Used
- Amazon S3
- Static Website Hosting
- Bucket Policy

## Steps
1. Created an S3 bucket
2. Uploaded index.html
3. Enabled static website hosting
4. Configured public bucket policy
5. Accessed the website endpoint

## Result
A live static website hosted on AWS S3.
