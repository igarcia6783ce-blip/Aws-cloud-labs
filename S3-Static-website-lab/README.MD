# AWS S3 Static Website Hosting Lab

## Overview
This lab demonstrates hosting a static website using Amazon S3.

## Architecture

User Browser
↓
Amazon S3 Bucket
↓
Static Website Hosting

## AWS Services Used

- Amazon S3
- Bucket Policies
- Static Website Hosting

## Steps Performed

1. Created an S3 bucket
2. Disabled block public access
3. Uploaded index.html
4. Enabled static website hosting
5. Applied bucket policy to allow public read access
6. Tested website using S3 website endpoint

## Skills Demonstrated

- S3 bucket configuration
- Static website hosting
- Public access policies
- Cloud storage fundamentals
